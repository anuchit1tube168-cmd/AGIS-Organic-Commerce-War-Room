คอนเทนต์ที่สร้างแล้วจะมาอยู่ตรงนี้ กด Copy เอาไปโพสต์ได้เลย

สูตรโพสต์ที่ใช้ได้ทันที

สูตร 1: ปัญหา → ทางออก → CTA
คุณเคยเจอปัญหา [Pain] ไหม? ตัวนี้ช่วยให้ [Result] เหมาะกับ [Target] สนใจดูรายละเอียดที่ลิงก์
สูตร 2: ก่อนซื้อควรรู้
ก่อนซื้อ [Product] ต้องดู 3 จุดนี้: ราคา, การใช้งานจริง, เหมาะกับใคร ถ้าใช่ค่อยกด
สูตร 3: รีวิวแบบไม่อวย
ลองใช้แล้วชอบตรง [Benefit] แต่ข้อเสียคือ [Limitation] ถ้ารับได้ ถือว่าคุ้ม
สูตร 4: โปรโมชัน
ถ้ากำลังหา [Product] ช่วงนี้มี [Offer] ราคา [Price] ลองเช็คก่อนของหมด

API / Token ใช้ทีหลัง

ตอนนี้ใช้ Manual ก่อน: ระบบเจนคอนเทนต์ → Copy ไปโพสต์เอง → กลับมากรอกยอด เมื่อเจอสินค้าชนะแล้วค่อยต่อ API จริงทีละตัว ห้ามใส่ token จริงในหน้าเว็บ

Platformสิ่งที่ต้องใช้จริงในอนาคตเริ่มเมื่อไหร่
OpenAI/AIAPI Key ผ่าน Backend เท่านั้นเมื่ออยากให้ AI เจนจริงบนเว็บ
Facebook/InstagramMeta App, Page Token, IG permissionsเมื่อมีโพสต์ที่ชนะแล้ว
YouTubeGoogle OAuth, Refresh Token, Channel IDเมื่อทำ Shorts ต่อเนื่อง
Shopee/LazadaAffiliate link / partner key / shop tokenเมื่อเริ่มมีออเดอร์
TikTokClient Key/Secret, OAuth, Content Posting permissionหลังระบบ manual ทำเงินแล้ว

Prototype Safe Mode: ข้อมูลเก็บในเครื่องผู้ใช้ด้วย localStorage เท่านั้น ถ้าจะใช้งานจริงระดับทีมให้ต่อ Supabase/Firebase และ Backend สำหรับซ่อน secret key
