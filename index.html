API / Token Strategy

MVP นี้ยังไม่ใช้ token จริง ให้ใช้แบบ Manual ก่อน: สร้างโพสต์ → กด Copy → ไปโพสต์เอง → เอายอดกลับมากรอก เมื่อเจอสินค้าชนะแล้วค่อยต่อ API ทีละตัว

PlatformToken ที่ต้องใช้จริงPhase
OpenAI/AIAPI Key ผ่าน Backend เท่านั้นPhase 2
Facebook/InstagramMeta App, Page Token, IG permissionsPhase 3
YouTubeGoogle OAuth, Refresh Token, Channel IDPhase 3
Shopee/LazadaPartner/App Key, Shop Token, Affiliate LinkPhase 2–4
TikTokClient Key/Secret, OAuth, Content Posting permissionPhase 4

Prototype Safe Mode: เก็บข้อมูลใน browser memory เท่านั้น รีเฟรชแล้วข้อมูล demo อาจหาย หากใช้จริงให้ต่อ Supabase/Firebase และเก็บ secret ใน backend ห้ามใส่ key จริงในหน้าเว็บ
