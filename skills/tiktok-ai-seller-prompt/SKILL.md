# TikTok AI Seller Prompt Skill

> สกิลสำหรับถอดไอเดียจากคลิป TikTok/Short/Reels แล้วแปลงเป็นระบบทำคอนเทนต์ขายของแบบใช้งานจริง: คิดมุมขาย, เขียนสคริปต์, ทำ Hook, CTA, Caption, Hashtag, Prompt สำหรับ AI และ Checklist ก่อนโพสต์

---

## 1. เป้าหมายของสกิล

ใช้เมื่อผู้ใช้ส่งลิงก์ TikTok, Reels, Shorts หรือข้อความ/ภาพตัวอย่างคอนเทนต์ แล้วต้องการให้ AI “ถอดสูตร” ออกมาเป็นระบบที่นำไปใช้ซ้ำได้ โดยเฉพาะงานต่อไปนี้

- ทำคลิปขายของ TikTok / Shopee / Facebook / YouTube Shorts
- สร้าง Prompt สำหรับให้ AI ช่วยคิดคอนเทนต์
- ถอดโครงสร้างคลิปไวรัลเป็น Template
- ทำสคริปต์คลิป 15–60 วินาที
- สร้าง Caption, Hashtag, CTA และแนวทางยิงซ้ำหลายเวอร์ชัน
- ทำระบบคอนเทนต์สำหรับ Affiliate / ร้านค้า / สินค้าออนไลน์

---

## 2. หลักคิดของสกิล

คอนเทนต์ขายของที่ดีไม่ใช่แค่ “พูดว่าสินค้าดี” แต่ต้องมี 6 ชั้น

1. **Attention** — เปิดคลิปให้คนหยุดดูใน 1–3 วินาที
2. **Problem** — จี้ปัญหาหรือความต้องการของกลุ่มเป้าหมาย
3. **Insight** — ทำให้คนรู้สึกว่า “เออ จริง”
4. **Product Bridge** — เชื่อมสินค้าเข้าไปแบบไม่ฝืน
5. **Proof / Demo** — มีหลักฐาน ทดลองใช้ เปรียบเทียบ หรือผลลัพธ์
6. **Action** — บอกให้ทำอะไรต่อ เช่น กดตะกร้า ทักแชต กดลิงก์ บันทึกไว้

---

## 3. Input ที่ต้องรับจากผู้ใช้

ถามให้น้อยที่สุด ถ้าข้อมูลไม่ครบให้เดาแบบมีเหตุผล และแจ้งสมมติฐาน

```text
สินค้า/บริการ:
กลุ่มเป้าหมาย:
แพลตฟอร์มที่จะลง:
เป้าหมายของคลิป: ขาย / ปักตะกร้า / สมัคร / เก็บ lead / สร้างแบรนด์
โทน: จริงใจ / สนุก / มืออาชีพ / ทหาร / หรู / บ้าน ๆ / รีวิวจริง
ความยาวคลิป: 15 / 30 / 45 / 60 วินาที
ข้อห้าม: คำเคลมเกินจริง, การแพทย์, การเงิน, ก่อน-หลัง, รับประกันผล
```

ถ้าผู้ใช้ส่งแค่ลิงก์ ให้ตอบแบบนี้ก่อนทำงาน:

```text
ผมจะถอดเป็น 5 ส่วน: สูตรคอนเทนต์, Hook, Script, Prompt ใช้ซ้ำ, และ Checklist ก่อนโพสต์
ถ้าดึงเนื้อหาจากลิงก์ไม่ได้ ผมจะใช้ข้อมูลที่เห็น/ข้อความที่แนบ/บริบทของผู้ใช้ แล้วทำเป็น Template ใช้งานจริงให้ก่อน
```

## 4. Workflow การถอดสกิลจากคลิป

### Step 1 — Identify Content Type

จัดประเภทคลิปก่อนว่าเป็นแบบใด

- รีวิวสินค้า
- สอนใช้เครื่องมือ AI
- เปรียบเทียบก่อน–หลัง
- เล่าเคสตัวอย่าง
- ขายตรงพร้อม CTA
- สร้างความรู้ก่อนขาย
- Affiliate / ปักตะกร้า
- Personal brand / Expert content

### Step 2 — Extract Viral Structure

ถอดโครงสร้างคลิปเป็นตาราง

| ช่วงเวลา | หน้าที่ของช่วงนั้น | สิ่งที่พูด/แสดง | เหตุผลที่ทำให้คนดูต่อ |
|---|---|---|---|
| 0–3 วิ | Hook | ประโยคเปิด/ภาพเปิด | หยุดนิ้ว |
| 4–10 วิ | Problem | ปัญหา/ความสงสัย | ทำให้เกี่ยวกับคนดู |
| 11–25 วิ | Value | วิธี/สินค้า/ตัวอย่าง | ให้ประโยชน์ |
| 26–45 วิ | Proof | ทดลอง/ผลลัพธ์/รีวิว | เพิ่มความน่าเชื่อถือ |
| ท้ายคลิป | CTA | กดตะกร้า/ทักแชต/ติดตาม | ปิดการกระทำ |

### Step 3 — Convert to Prompt System

แปลงคลิปเป็น Prompt ใช้ซ้ำ ไม่ใช่แค่สรุปคลิป

```text
คุณคือ TikTok Commerce Strategist
ช่วยสร้างคอนเทนต์ขายสินค้าแบบ Organic สำหรับ [สินค้า]
กลุ่มเป้าหมายคือ [กลุ่มเป้าหมาย]
เป้าหมายคือ [ขาย/ปักตะกร้า/เก็บ lead]
ให้สร้างผลลัพธ์เป็น:
1) Hook 10 แบบ
2) Script คลิป 30 วินาที 3 เวอร์ชัน
3) Caption 5 แบบ
4) CTA 5 แบบ
5) Hashtag 20 คำ
6) Checklist ก่อนโพสต์
ใช้ภาษาไทย กระชับ เป็นธรรมชาติ ไม่ขายแข็งเกินไป
ห้ามเคลมเกินจริง และต้องมีมุมประโยชน์ชัดเจน
```

## 5. Prompt หลักสำหรับใช้งานจริง

### Prompt A — ถอดคลิปเป็นสูตร

```text
ถอดสูตรคอนเทนต์จากคลิป/ข้อความนี้ให้เป็นระบบใช้งานซ้ำได้

ข้อมูลต้นทาง:
[วางลิงก์/ข้อความ/ภาพ/สรุปคลิป]

ให้วิเคราะห์เป็น:
1. ประเภทคอนเทนต์
2. กลุ่มเป้าหมายที่คลิปนี้จับ
3. Hook ที่ใช้
4. Pain point ที่จี้
5. Value ที่ให้
6. วิธีเชื่อมสินค้า/บริการ
7. CTA
8. จุดที่ทำให้คนอยากดูต่อ
9. สิ่งที่ควรเลียนแบบ
10. สิ่งที่ไม่ควรเลียนแบบ
11. Template สำหรับทำคลิปใหม่ 5 แบบ
12. Prompt สำหรับให้ AI สร้างคลิปแนวเดียวกัน

ตอบเป็นภาษาไทยแบบพร้อมใช้งาน
```

### Prompt B — ทำสคริปต์ขายของ TikTok 30 วินาที

```text
สร้างสคริปต์ TikTok 30 วินาทีสำหรับขายสินค้า:
สินค้า: [ชื่อสินค้า]
กลุ่มเป้าหมาย: [ใคร]
ปัญหาหลัก: [ปัญหา]
จุดขาย: [จุดขาย]
ข้อจำกัด/ข้อห้าม: [ข้อห้าม]
โทน: [จริงใจ/สนุก/มืออาชีพ]

ขอผลลัพธ์เป็น:
- Hook 3 วิแรก 5 แบบ
- Script แบบพูดหน้ากล้อง 3 เวอร์ชัน
- Shot list ว่าควรถ่ายภาพอะไรบ้าง
- Caption 3 แบบ
- CTA 3 แบบ
- Hashtag 20 คำ
```

### Prompt C — ทำ 100 ไอเดียคลิปต่อวัน

```text
คุณคือทีม Content War Room สำหรับ TikTok Commerce
ช่วยแตกไอเดียคลิป 100 ไอเดียให้กับสินค้า [สินค้า]
โดยแบ่งเป็นหมวด:
1. Problem/Pain point
2. How-to
3. Review
4. Compare
5. Myth vs Fact
6. Mistake
7. Before/After แบบไม่เคลมเกินจริง
8. Storytelling

10. Trend adaptation

แต่ละไอเดียให้มี:
- ชื่อคลิป
- Hook
- โครงสคริปต์ 3 บรรทัด
- CTA
- เหมาะกับความยาวกี่วินาที

ใช้ภาษาไทย เข้าใจง่าย ทำได้จริง
```

### Prompt D — แปลงสินค้าเป็นมุมขาย

```text
ช่วยวิเคราะห์สินค้า [ชื่อสินค้า]
รายละเอียดสินค้า: [รายละเอียด]
ราคา: [ราคา]
ลูกค้าเป้าหมาย: [กลุ่มเป้าหมาย]
คู่แข่ง/สินค้าคล้ายกัน: [ถ้ามี]

ขอผลลัพธ์เป็น:
1. มุมขายหลัก 10 มุม
2. Pain point 20 ข้อ
3. Benefit ที่พูดได้แบบไม่เว่อร์ 20 ข้อ
4. Objection ลูกค้า 10 ข้อ พร้อมคำตอบ
5. Hook 30 แบบ
6. Script คลิป 15 วิ / 30 วิ / 60 วิ
7. Caption พร้อม CTA
8. ไอเดียภาพประกอบ/ช็อตที่ต้องถ่าย
```

## 7. Hook Bank สำหรับคลิปขายของ

เลือกใช้ให้ตรงกับสินค้า

### Hook แบบจี้ปัญหา

- ถ้าคุณเจอปัญหานี้อยู่ ห้ามเลื่อนผ่าน
- คนส่วนใหญ่พลาดตรงนี้โดยไม่รู้ตัว
- ก่อนซื้อ [สินค้า] ต้องรู้เรื่องนี้
- ใช้ [สินค้า] ผิด ชีวิตเหนื่อยกว่าเดิม
- 3 สิ่งที่คนซื้อ [สินค้า] มักไม่เคยเช็ก

### Hook แบบผลลัพธ์

- ลองใช้วิธีนี้ 7 วัน ผลคือ...
- จากที่เคยเสียเวลา ตอนนี้เหลือแค่นี้
- วิธีที่ช่วยให้ [ผลลัพธ์] ง่ายขึ้น
- ไม่ต้องเก่งก็เริ่มได้ แค่ทำตามนี้
- ตัวนี้แก้ปัญหาที่ผมเจอมานาน

### Hook แบบเปรียบเทียบ

- ก่อนซื้อรุ่นแพง ดูตัวนี้ก่อน
- อันนี้เหมือนกันตรงไหน ต่างกันตรงไหน
- จ่ายถูกกว่า แต่ได้อะไรบ้าง
- ซื้อผิดชีวิตเปลี่ยน เทียบให้ดูชัด ๆ
- ตัวไหนเหมาะกับใคร ดูจบเลือกได้เลย

### Hook แบบ AI/เครื่องมือ

- ใช้ AI ช่วยขายของแบบนี้ได้เลย
- Prompt นี้ช่วยคิดคลิปได้ทั้งเดือน
- ไม่ต้องคิดคอนเทนต์เอง เริ่มจากคำสั่งนี้
- เอา ChatGPT มาช่วยทำ TikTok Shop แบบนี้
- สูตรทำคลิปขายของด้วย AI ที่ใช้ซ้ำได้

## 8. Checklist ก่อนโพสต์

- [ ] 3 วินาทีแรกชัดว่าคลิปนี้เกี่ยวกับใคร
- [ ] มีปัญหาหรือประโยชน์ที่จับต้องได้
- [ ] ไม่เคลมเกินจริง
- [ ] มีตัวอย่าง/ภาพ/หลักฐานประกอบ
- [ ] CTA ชัดเจน
- [ ] Caption ไม่ยาวเกินไป
- [ ] Hashtag มีทั้งคำกว้างและคำเฉพาะ
- [ ] ภาพปกอ่านออกบนมือถือ
- [ ] เสียงพูดชัด
- [ ] คลิปไม่วนขายแข็งจนคนหนี

## 9. Guardrails / ข้อควรระวัง

ห้ามสร้างเนื้อหาที่:

- เคลมผลลัพธ์เกินจริง เช่น รวยทันที ผอมทันที หายป่วยแน่นอน
- หลอกให้เข้าใจผิดเรื่องราคา/โปรโมชัน
- ใช้รีวิวปลอม
- ใช้คำทางการแพทย์/การเงินโดยไม่มีหลักฐาน
- ขายสินค้าอันตราย ผิดกฎหมาย หรือมีข้อจำกัดแพลตฟอร์ม
- คัดลอกสคริปต์คนอื่นแบบตรงตัว ควรถอดเป็น “สูตร” แล้วเขียนใหม่

## 10. Agent Mode

เมื่อต้องทำงานเป็นทีม ให้แบ่ง Agent แบบนี้

1. **Trend Scout** — หาเทรนด์และมุมคลิป
2. **Audience Analyst** — วิเคราะห์กลุ่มเป้าหมายและ pain point
3. **Prompt Engineer** — เขียน prompt ใช้ซ้ำ
4. **Script Writer** — เขียนสคริปต์คลิป
5. **Commerce Strategist** — วาง CTA และมุมขาย
6. **Compliance Checker** — ตรวจคำเคลมและความเสี่ยง
7. **Posting Planner** — จัดตารางลงคลิป
