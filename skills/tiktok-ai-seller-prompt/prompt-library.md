## 1. Master Prompt — ถอดสูตรจากคลิป

```text
คุณคือ TikTok Commerce Strategist และ Prompt Engineer
หน้าที่คือถอดสูตรจากคลิป/ข้อความต้นทาง แล้วแปลงเป็นระบบทำคอนเทนต์ขายของที่ใช้ซ้ำได้

ข้อมูลต้นทาง:
[วางลิงก์/ข้อความ/ภาพ/สรุปคลิป]

บริบทสินค้า:
สินค้า:
กลุ่มเป้าหมาย:
แพลตฟอร์ม:
เป้าหมาย: ขาย / ปักตะกร้า / เก็บ lead / สร้างแบรนด์
โทนภาษา:
ข้อห้าม:

ขอผลลัพธ์เป็น:
1. แก่นของคลิป
2. โครงสร้างไวรัล 0–3 / 4–10 / 11–25 / 26–45 / ท้ายคลิป
3. Hook ที่ใช้
4. Pain point ที่จี้
5. Value ที่ให้
6. Product Bridge
7. Proof / Demo
8. CTA

12. Template คลิปใหม่ 5 แบบ
13. Script 30 วินาที 3 เวอร์ชัน
14. Caption 5 แบบ
15. Hashtag 20 คำ
ตอบเป็นภาษาไทย ใช้งานได้จริง ไม่ขายฝัน ไม่เคลมเกินจริง
```

```text
ช่วยแตกไอเดียคลิป 30 คลิปจากสินค้าเดียวนี้

- Caption
```

## 4. Prompt — Hook Generator

```text
ช่วยสร้าง Hook 50 แบบสำหรับคลิปขายสินค้า [สินค้า]
กลุ่มเป้าหมายคือ [กลุ่มเป้าหมาย]
ปัญหาหลักคือ [ปัญหา]

แบ่ง Hook เป็น:
1. จี้ปัญหา 10 แบบ
2. เปรียบเทียบ 10 แบบ
3. เตือนก่อนซื้อ 10 แบบ
4. รีวิวจริงใจ 10 แบบ
5. AI/เทคนิค/สูตรลับ 10 แบบ

เงื่อนไข:
- อ่านแล้วหยุดนิ้วใน 3 วินาที
- ไม่เคลมเกินจริง
- ไม่หลอกลวง
- ภาษาไทยธรรมชาติ
```

## 5. Prompt — Script 15/30/60 วินาที

```text
สร้าง Script คลิปขายสินค้า [สินค้า]
กลุ่มเป้าหมาย [กลุ่มเป้าหมาย]
จุดขาย [จุดขาย]
Pain point [ปัญหา]
CTA [กดตะกร้า/ทักแชต/กดลิงก์/ติดตาม]

ขอ 3 ความยาว:
1. 15 วินาที — เร็ว กระแทก ตรงประเด็น
2. 30 วินาที — อธิบายครบ มีตัวอย่าง
3. 60 วินาที — เล่าเรื่อง + demo + ปิดขาย

แต่ละเวอร์ชันให้แยก:
- คำพูดหน้ากล้อง
- ภาพที่ควรถ่าย
- ข้อความบนจอ
- เสียง/จังหวะ
- CTA ท้ายคลิป
```
