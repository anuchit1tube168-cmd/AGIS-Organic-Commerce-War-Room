# SOP — TikTok AI Seller Content War Room

เอกสารนี้คือขั้นตอนปฏิบัติงานมาตรฐานสำหรับผลิตคอนเทนต์ขายของด้วย AI ตั้งแต่รับสินค้า → ถอดมุมขาย → เขียนสคริปต์ → ถ่าย → ตรวจ → โพสต์ → วิเคราะห์ผล

---

## 1. เป้าหมายของ SOP

สร้างระบบที่ทำให้คนเดียวหรือทีมเล็กสามารถผลิตคลิปขายของได้อย่างต่อเนื่อง โดยยังควบคุมคุณภาพ ความถูกต้อง และความเสี่ยงของคำเคลม

ผลลัพธ์ที่ต้องได้ในแต่ละวัน:

- ไอเดียคลิปพร้อมถ่าย
- Script พร้อมพูด
- Shot list พร้อมถ่าย
- Caption / CTA / Hashtag พร้อมโพสต์
- Checklist ตรวจความเสี่ยง
- ตารางวัดผลหลังโพสต์

---

## 2. บทบาทในทีม

ถ้าทำคนเดียว ให้ใช้ AI จำลองบทบาททั้งหมด

| บทบาท | หน้าที่ | Output |
|---|---|---|
| Product Owner | ให้ข้อมูลสินค้า ราคา จุดขาย ข้อห้าม | Product Brief |
| Trend Scout | หาเทรนด์/มุมไวรัล | Trend Angle |
| Audience Analyst | วิเคราะห์ลูกค้าและ pain point | Audience Map |
| Script Writer | เขียนสคริปต์ | Script 15/30/60 วิ |
| Shooter | ถ่ายคลิปตาม shot list | Raw Clips |
| Editor | ตัดต่อ ใส่ subtitle ปก | Final Video |
| Compliance Checker | ตรวจคำเคลม/ความเสี่ยง | Risk Report |
| Performance Analyst | วิเคราะห์ผลหลังโพสต์ | Optimization Plan |

## 3. Workflow รายวัน

### Phase 1 — เตรียมข้อมูลสินค้า

กรอกข้อมูลใน `templates/product-brief-template.md`

ขั้นต่ำต้องมี:

- ชื่อสินค้า
- ราคา
- กลุ่มเป้าหมาย
- Pain point
- จุดขาย
- ข้อห้ามในการสื่อสาร
- CTA ที่ต้องการ

### Phase 2 — แตกมุมขาย

ใช้ Prompt:

```text
ใช้สกิล tiktok-ai-seller-prompt วิเคราะห์ Product Brief นี้ แล้วแตกมุมขาย 30 คลิป แบ่งเป็น Problem / How-to / Review / Compare / FAQ / Storytelling
```

### Phase 3 — คัดไอเดียที่จะถ่ายวันนี้

เกณฑ์คัดเลือก:

- ทำง่าย
- Hook ชัด
- เห็นสินค้าในคลิป
- มีประโยชน์จริง
- ไม่เสี่ยงเคลมเกินจริง
- CTA ชัด

ให้ให้คะแนน 1–5 ใน 3 ช่อง:

| คลิป | ผลต่อยอดขาย | ความง่าย | ความเสี่ยง | สรุป |
|---|---:|---:|---:|---|
| คลิป 1 | 5 | 4 | 1 | ถ่ายก่อน |

### Phase 4 — เขียน Script และ Shot List

ทุกคลิปต้องมี 5 ส่วน:

1. Hook 0–3 วิ
2. Problem / Situation
3. Product Bridge
4. Demo / Proof
5. CTA

ตัวอย่างโครง:

```text
[Hook]
ถ้าคุณกำลังเจอปัญหา _______ อย่าเพิ่งซื้อ _______ จนกว่าจะดูคลิปนี้

[Problem]
หลายคนคิดว่า _______ แต่จริง ๆ แล้ว _______

[Bridge]
ตัวนี้ช่วยตรงที่ _______

[Demo]
ดูตรงนี้ ผมลองให้เห็นแบบง่าย ๆ _______

[CTA]
ถ้าอยากดูรายละเอียด กดตะกร้า/ทักแชตไว้ได้เลย
```

### Phase 5 — ถ่ายทำ

Checklist ก่อนถ่าย:

- [ ] แสงสว่างพอ
- [ ] เสียงชัด
- [ ] สินค้าอยู่ในเฟรม
- [ ] มีภาพใกล้สินค้า
- [ ] มีภาพการใช้งานจริง
- [ ] ถ่าย Hook หลายเวอร์ชัน
- [ ] ถ่าย B-roll เผื่อตัดต่อ
- [ ] พูด CTA แยกไว้หลายแบบ

### Phase 6 — ตัดต่อ

มาตรฐานตัดต่อ:

- เปิดด้วยภาพ/ข้อความที่หยุดนิ้ว
- Subtitle อ่านง่าย
- ตัดช่วงเงียบออก
- ไม่ใส่เอฟเฟกต์เยอะจนกลบสินค้า
- ใส่ข้อความบนจอเฉพาะประเด็นสำคัญ
- ปิดด้วย CTA ชัดเจน

### Phase 7 — ตรวจ Compliance

ใช้ `guardrails.md` ก่อนโพสต์ทุกครั้ง โดยเฉพาะสินค้ากลุ่ม:

- สุขภาพ/ความงาม
- การเงิน/ลงทุน
- อาหารเสริม
- เด็ก/ผู้สูงอายุ
- อุปกรณ์ที่เกี่ยวกับความปลอดภัย

### Phase 8 — โพสต์และบันทึกผล

หลังโพสต์ 24 ชั่วโมง บันทึก:

- ยอดวิว
- Watch time
- Completion rate
- Like
- Comment
- Share
- CTR / คลิกตะกร้า
- ยอดขาย
- คอมเมนต์ที่เจอบ่อย

### Phase 9 — วิเคราะห์และวนซ้ำ

กฎง่าย ๆ:

- วิวดีแต่ขายไม่ดี → Hook ดี แต่ CTA/Product Bridge อ่อน
- วิวน้อยแต่คนซื้อ → ทำ Hook ใหม่และยิงซ้ำ
- คนคอมเมนต์ถามเยอะ → ทำ FAQ Clip ต่อทันที
- Completion ต่ำ → คลิปยาว/เข้าประเด็นช้า
- Share สูง → ทำซีรีส์ต่อ

## 4. Daily Production Standard

### Minimum Mode — คนเดียว มือถือเครื่องเดียว

เป้าหมายต่อวัน:

- 10 ไอเดีย
- 5 Script
- 3 คลิปถ่ายจริง
- 1–3 คลิปโพสต์

### Pro Mode — ทีมเล็ก

เป้าหมายต่อวัน:

- 30 ไอเดีย
- 15 Script
- 10 คลิปถ่ายจริง
- 5–10 คลิปโพสต์

### War Room Mode — ทำจริงจัง

เป้าหมายต่อวัน:

- 100 ไอเดีย
- 30 Script
- 20 คลิปถ่ายจริง
- 10–20 คลิปโพสต์
- วิเคราะห์ผลทุก 24 ชม.

---

## 5. Quality Gate ก่อนโพสต์

คลิปต้องผ่าน 7 ข้อนี้ก่อนลง

- [ ] Hook เข้าใจใน 3 วิ
- [ ] รู้ชัดว่าคลิปนี้เหมาะกับใคร
- [ ] มีภาพสินค้า/วิธีใช้
- [ ] ไม่ขายแข็งจนเกินไป
- [ ] ไม่มีคำเคลมเกินจริง
- [ ] CTA ชัด
- [ ] Caption ไม่ขัดกับเนื้อหาคลิป

ถ้าไม่ผ่าน ให้แก้ก่อนโพสต์ อย่าฝืนลงเพราะ “เสียดายคลิป” คลิปที่ไม่ชัดคือภาษีเวลา

---

## 6. Weekly Review

ทุก 7 วัน ให้สรุป:

| หมวด | จำนวนคลิป | วิวเฉลี่ย | CTR | ยอดขาย | ควรทำต่อไหม |
|---|---:|---:|---:|---:|---|
| Problem | | | | | |
| How-to | | | | | |
| Review | | | | | |
| Compare | | | | | |
| FAQ | | | | | |

คำถามที่ต้องตอบ:

1. Hook แบบไหนชนะ
2. Pain point ไหนขายดีที่สุด
3. CTA ไหนทำให้คลิกมากที่สุด
4. ความยาวคลิปเท่าไหร่เหมาะสุด
5. สินค้าควรปรับมุมขายไหม
6. ควรทำซีรีส์อะไรต่อ

## 7. Naming Convention

ใช้ชื่อไฟล์แบบนี้เพื่อจัดระบบ

```text
YYYYMMDD_product_angle_version_platform
```

ตัวอย่าง:

```text
20260627_waterbottle_problem_v01_tiktok
20260627_waterbottle_review_v02_shopee
20260627_waterbottle_faq_v01_reels
```

## 8. Done Definition

งาน 1 คลิปถือว่าเสร็จเมื่อมีครบ:

- Script
- Shot list
- Video file
- Caption
- Hashtag
- CTA
- Thumbnail/cover text
- Compliance check
- Performance record หลังโพสต์
