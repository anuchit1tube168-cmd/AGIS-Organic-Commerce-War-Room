# Guardrails — TikTok AI Seller Prompt

เอกสารนี้ใช้ตรวจความเสี่ยงก่อนโพสต์คอนเทนต์ขายของ โดยเฉพาะคำเคลมที่อาจเกินจริง ทำให้เข้าใจผิด หรือเสี่ยงผิดนโยบายแพลตฟอร์ม

---

## 1. หลักคิด

ขายได้ไม่จำเป็นต้องพูดเว่อร์ คอนเทนต์ที่ดีต้องทำให้คนเชื่อจากเหตุผล ตัวอย่าง และประสบการณ์ ไม่ใช่สัญญาผลลัพธ์เกินจริง

---

## 2. คำที่ต้องระวัง

| คำ/แนวพูด | ความเสี่ยง | วิธีแก้ให้ปลอดภัยขึ้น |
|---|---|---|
| หายแน่นอน | เคลมผลรักษา | อาจช่วยให้รู้สึกดีขึ้น / ขึ้นกับแต่ละบุคคล |
| รวยทันที | เคลมการเงิน | เป็นแนวทาง/ตัวอย่าง ไม่รับประกันผลลัพธ์ |
| ผอมใน 7 วัน | เคลมสุขภาพ/รูปร่าง | ใช้ประกอบการดูแลตัวเอง ผลลัพธ์ขึ้นกับแต่ละบุคคล |
| ดีที่สุดในโลก | เกินจริง | เป็นหนึ่งในตัวเลือกที่น่าสนใจ |
| ซื้อแล้วชีวิตเปลี่ยน | โฆษณาเกินจริง | อาจช่วยให้การใช้งานสะดวกขึ้น |
| การันตี 100% | รับประกันเกินจริง | เงื่อนไขเป็นไปตามที่ร้านกำหนด |
| ไม่ต้องทำอะไรเลย | ทำให้เข้าใจผิด | ช่วยลดขั้นตอนบางส่วน |
| เห็นผลทันทีทุกคน | เคลมผลรวม | ผลลัพธ์ขึ้นกับการใช้งานและแต่ละบุคคล |

---

## 3. สินค้ากลุ่มเสี่ยงสูง

ต้องตรวจเข้มเป็นพิเศษถ้าเป็นสินค้าเหล่านี้

- อาหารเสริม
- สุขภาพ/ความงาม
- สกินแคร์
- อุปกรณ์การแพทย์
- การเงิน/ลงทุน/คริปโต
- คอร์สสอนหารายได้
- สินค้าเด็ก/ผู้สูงอายุ
- อุปกรณ์ความปลอดภัย
- สินค้าที่ต้องมีใบอนุญาตหรือข้อจำกัด

---

## 4. Checklist ก่อนโพสต์

### ด้านความจริงของข้อมูล

- [ ] ราคาและโปรโมชันถูกต้อง
- [ ] วิธีใช้ไม่ทำให้เข้าใจผิด
- [ ] สรรพคุณไม่เกินข้อมูลจริง
- [ ] ไม่มีการอ้างผลลัพธ์ที่พิสูจน์ไม่ได้
- [ ] ถ้ามีรีวิว ต้องเป็นรีวิวจริงหรือระบุว่าเป็นตัวอย่าง

### ด้านภาษา

- [ ] ไม่มีคำว่า หายแน่นอน / รวยแน่นอน / ผอมแน่นอน
- [ ] ไม่มีคำว่า 100% ถ้าไม่มีเงื่อนไขชัด
- [ ] ไม่มีการโจมตีคู่แข่งแบบเสียหาย
- [ ] ไม่มีการใช้ความกลัวเกินจำเป็น
- [ ] ไม่มีคำสั่งให้รีบซื้อโดยหลอกว่าของหมด ถ้าไม่จริง

### ด้านภาพ/วิดีโอ

- [ ] ภาพก่อน–หลังไม่หลอกลวง
- [ ] Demo ไม่ตัดต่อให้เข้าใจผิด
- [ ] ภาพสินค้าเป็นสินค้าจริง
- [ ] Subtitle ไม่ขัดกับคำพูด
- [ ] ปกคลิปไม่เคลมเกินจริง

### ด้าน CTA

- [ ] CTA ไม่บังคับหรือหลอกลวง
- [ ] เงื่อนไขโปรชัดเจน
- [ ] ไม่อ้างว่ามีจำนวนจำกัดถ้าไม่จริง
- [ ] ไม่ใช้คำว่า “รับประกันผล” ถ้าไม่มีเงื่อนไข

---

## 5. ระดับความเสี่ยง

| ระดับ | ความหมาย | การจัดการ |
|---|---|---|
| ต่ำ | ใช้ได้ แต่อาจแก้ให้นุ่มขึ้น | โพสต์ได้ |
| กลาง | มีคำที่อาจตีความเกินจริง | แก้ก่อนโพสต์ |
| สูง | มีโอกาสผิดนโยบายหรือหลอกลวง | ห้ามโพสต์จนกว่าจะแก้ |

---

## 6. สูตรแก้คำเคลม

### จากเคลมแรง → เคลมปลอดภัย

```text
หายแน่นอน → อาจช่วยบรรเทา / ช่วยให้ดูแลง่ายขึ้น
รวยทันที → เป็นแนวทางเพิ่มโอกาส / ต้องใช้เวลาและการลงมือทำ
ผอมไว → ช่วยเป็นส่วนหนึ่งของการดูแลรูปร่าง
ดีที่สุด → เป็นตัวเลือกที่น่าสนใจสำหรับคนที่ต้องการ...
ใช้แล้วเห็นผลทุกคน → ผลลัพธ์ขึ้นกับแต่ละบุคคล
ไม่ต้องทำอะไรเลย → ช่วยลดขั้นตอนบางส่วน
```

---

## 7. Prompt ตรวจความเสี่ยง

```text
ตรวจคอนเทนต์นี้ก่อนโพสต์

Script:
[วาง Script]

Caption:
[วาง Caption]

CTA:
[วาง CTA]

สินค้า:
[รายละเอียดสินค้า]

ช่วยตรวจเป็นตาราง:
1. ประโยคที่เสี่ยง
2. ประเภทความเสี่ยง
3. ระดับ ต่ำ/กลาง/สูง
4. เหตุผล
5. ประโยคที่แก้แล้ว
6. ผ่าน/ไม่ผ่าน

เป้าหมายคือยังขายได้ แต่ปลอดภัยขึ้น
```

---

## 8. ตัวอย่างการแก้ Script

### ก่อนแก้

```text
ตัวนี้ช่วยให้คุณขายดีทันที แค่กดใช้ก็มีออเดอร์เข้าแน่นอน
```

### หลังแก้

```text
ตัวนี้ช่วยจัดระบบคอนเทนต์และทำให้คุณมีโอกาสสร้างออเดอร์ได้ง่ายขึ้น ถ้าใช้ต่อเนื่องและเลือกสินค้าถูกกลุ่ม
```
