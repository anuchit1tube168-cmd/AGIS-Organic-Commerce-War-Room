# Sample Output — TikTok AI Seller Prompt

ตัวอย่างนี้ใช้จำลองผลลัพธ์เมื่อกรอก Product Brief แล้วให้ AI แตกคอนเทนต์ขายของ

---

## สินค้าตัวอย่าง

```text
สินค้า: กระบอกน้ำเก็บอุณหภูมิ
กลุ่มเป้าหมาย: คนทำงาน นักเรียน นักศึกษา คนออกกำลังกาย
จุดขาย: เก็บเย็นนาน พกง่าย ดีไซน์เรียบ ราคาไม่แรง
CTA: กดตะกร้า
ข้อห้าม: ห้ามเคลมว่าเก็บเย็นได้แน่นอนทุกสภาพอากาศถ้าไม่มีผลทดสอบ
```

---

## 1. Audience Map

### กลุ่มเป้าหมายหลัก

คนที่ต้องพกน้ำไปทำงาน/เรียน/ออกกำลังกาย และเบื่อขวดน้ำที่ไม่เก็บเย็นหรือรั่วในกระเป๋า

### Pain Point

1. น้ำไม่เย็นระหว่างวัน
2. ขวดรั่วใส่กระเป๋า
3. ขวดใหญ่เกิน พกยาก
4. ดีไซน์ไม่เข้ากับโต๊ะทำงาน
5. ซื้อแล้วล้างยาก

### Desire

1. อยากมีน้ำเย็นดื่มทั้งวัน
2. อยากได้ของที่ดูดี
3. อยากได้ของใช้ประจำวันราคาไม่แรง

---

## 2. Hook 10 แบบ

1. ถ้าคุณพกน้ำไปทำงานทุกวัน ดูอันนี้ก่อน
2. ขวดน้ำที่ดีไม่ควรแค่ใส่น้ำได้ แต่มันต้องไม่สร้างปัญหา
3. เบื่อไหม น้ำเย็นตอนเช้า บ่ายกลายเป็นน้ำอุ่น
4. ก่อนซื้อกระบอกน้ำ เช็ก 3 จุดนี้ก่อน
5. กระบอกน้ำใบนี้เหมาะกับคนที่ไม่อยากซื้อน้ำขวดทุกวัน
6. ขวดน้ำรั่วในกระเป๋า ครั้งเดียวก็จำไปทั้งชีวิต
7. ของใช้เล็ก ๆ ที่ทำให้วันทำงานง่ายขึ้น
8. ไม่ต้องแพง แต่ต้องใช้จริงได้ทุกวัน
9. กระบอกน้ำที่วางบนโต๊ะแล้วไม่ขัดตา
10. ถ้าคุณเป็นสายทำงานนอกบ้าน ตัวนี้น่าสนใจ

---

## 3. Script 30 วินาที

### Version 1 — Problem Hook

```text
[0–3 วิ]
เบื่อไหมครับ น้ำเย็นตอนเช้า แต่บ่ายกลายเป็นน้ำอุ่น

[4–10 วิ]
หลายคนพกน้ำไปทำงาน แต่เจอปัญหาเดิม ๆ คือไม่เก็บเย็น หรือหนักเกินจนไม่อยากถือ

[11–22 วิ]
กระบอกน้ำตัวนี้ออกแบบมาให้พกง่าย วางบนโต๊ะก็ดูเรียบร้อย และช่วยเก็บอุณหภูมิได้ดีในชีวิตประจำวัน

[23–27 วิ]
ฝาปิดแน่น ใช้ในรถ ในออฟฟิศ หรือพกไปเรียนก็ได้

[28–30 วิ]
ใครกำลังหากระบอกน้ำไว้ใช้ทุกวัน กดตะกร้าดูรายละเอียดได้เลย
```

### On-screen Text

```text
น้ำเย็นตอนเช้า บ่ายกลายเป็นน้ำอุ่น?
กระบอกน้ำใช้ทุกวัน ต้องเลือกให้ถูก
พกง่าย | ดีไซน์เรียบ | ใช้ได้ทุกวัน
```

### Shot List

| เวลา | ภาพ |
|---|---|
| 0–3 วิ | เทน้ำเย็นลงแก้ว / ขวดน้ำวางบนโต๊ะ |
| 4–10 วิ | ขวดเก่ารั่ว/น้ำอุ่น ภาพจำลอง |
| 11–22 วิ | หมุนโชว์สินค้า ใส่น้ำ ปิดฝา |
| 23–27 วิ | ใส่กระเป๋า วางในรถ วางโต๊ะทำงาน |
| 28–30 วิ | ชี้ตะกร้า / ถือสินค้า |

## 4. Caption

### Caption สั้น

```text
กระบอกน้ำที่ใช้ได้ทุกวัน พกง่าย วางโต๊ะก็ดูดี กดตะกร้าดูรายละเอียดได้เลย
```

### Caption เล่าเรื่อง

```text
ของใช้เล็ก ๆ ที่ช่วยให้วันทำงานง่ายขึ้น ใครเบื่อน้ำไม่เย็นหรือขวดพกยาก ลองดูรุ่นนี้ไว้เป็นตัวเลือกครับ
```

### Caption ปิดขาย

```text
ใครกำลังหากระบอกน้ำไว้ใช้ประจำวัน รุ่นนี้พกง่าย ดีไซน์เรียบ ใช้ได้ทั้งเรียน ทำงาน และออกกำลังกาย กดตะกร้าได้เลย
```
